# Token Counter

A simple Node.js utility that counts tokens in various file formats using [Anthropic's tokenizer](https://github.com/anthropics/anthropic-tokenizer-typescript).

## Supported Formats

- PDF (`.pdf`)
- Word Documents (`.docx`)
- Excel Spreadsheets (`.xlsx`)
- CSV (`.csv`)
- Markdown (`.md`)
- Plain text (`.txt`)

## Usage

```bash
node count-tokens.js <filename>
```

## Installation

```bash
npm install
```
